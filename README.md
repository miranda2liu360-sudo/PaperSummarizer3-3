# CISC 101 – Research Paper Summarizer

This repository contains the final project for the Paper Summarizer system.

It includes:
- `system_prompt.md` – the main system prompt for the summarizer
- A `modules/` folder with:
  - Module 01 — Intake & Setup
  - Module 02 — Section Loop
  - Module 03 — Guardrails
  - Module 04 — Rendering & Refinement
  - Module 05 — Key Contributions Extractor
  - Module 06 — Citation / Reference Extractor

This repo is used for the CISC 101 Paper Summarizer assignment (Tasks 1 and 2).  
Task 3 updates to Module 2 and Module 3 are done on a separate branch and merged back into `main`.
