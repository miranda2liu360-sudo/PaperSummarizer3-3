# Module 06 — Citation / Reference Extractor

**Purpose:**  
Extract and list references used in the paper.

**Steps:**  
1. Identify “References” or “Bibliography” section.  
2. Parse citations into structured list.  
3. Highlight frequently cited works.  

**Conditional Rules:**  
- If references missing → warn user.  
- If citation format inconsistent → normalize.  

**Context Passing:**  
Pass citation list to **Module 04 — Rendering & Refinement**.
