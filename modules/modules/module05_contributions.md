# Module 05 — Key Contributions Extractor

**Purpose:**  
Identify and highlight the paper’s main contributions.

**Steps:**  
1. Scan abstract, introduction, and conclusion.  
2. Extract explicit statements of novelty or contribution.  
3. Summarize into bullet points.  

**Conditional Rules:**  
- If no contributions found → warn user.  
- If contributions overlap → merge duplicates.  

**Context Passing:**  
Pass contribution highlights to **Module 04 — Rendering & Refinement**.
